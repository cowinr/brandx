CLI reference

Code blocks render as plain monospace in email (no syntax colour) to avoid embedded span pollution in Outlook: 

brandx render report.md
brandx render --email report.md --clipboard
brandx render report.md --set colours.accent="#e63946"
 

"Identity is pure data, the engine is generic." The engine renders whatever resolved config it is handed; it hardcodes no person-specific value.
